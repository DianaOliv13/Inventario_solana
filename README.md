# Inventario_solana
Programa de inventario en Solana con Anchor. Permite crear, actualizar, activar/desactivar y eliminar productos en la blockchain.

## 📋 Descripción del Proyecto

**Inventario_solana** es una aplicación descentralizada (dApp) que implementa un sistema de gestión de inventario sobre la blockchain de Solana. Utiliza **Anchor Framework** para simplificar el desarrollo de programas Solana (smart contracts).

### Características principales:
- ✅ **Crear productos** - Registra nuevos productos en la blockchain
- ✏️ **Actualizar productos** - Modifica la información de productos existentes
- 🔄 **Activar/Desactivar productos** - Cambia el estado de disponibilidad de productos
- 🗑️ **Eliminar productos** - Remueve productos del inventario
- 🔐 **Seguridad en blockchain** - Todos los datos se almacenan de forma inmutable en Solana
